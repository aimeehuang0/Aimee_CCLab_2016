#Arduino Temparature Sensor

Two LED (RED & GREEN) state Change based on the Temparature

	Temp Below 19°C:                   RED↓ GREEN↓
	Temp Between 19-23°C(inclusive):   RED↑ GREEN↓
	Temp Above 23°C:                   RED↑ GREEN↑
